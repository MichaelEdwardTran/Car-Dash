This game was released by Michael Edward Tran in the summer of 2018 as "Car Dash" on the Android app store.

To try this game, extract the "Latest Build" folder and run "Builds.exe" found within.

Or, download it on the android app store at https://play.google.com/store/apps/details?id=com.TobikoGames.CarDash
